# Resume

LaTeX resume template.

## Build

```bash
pdflatex resume.tex
```

## Dev Container

VS Code dev container with LaTeX support:
- LaTeX Workshop extension with chktex linting and automatic formatting
- Pre-installed LaTeX packages: preprint, titlesec, marvosym, enumitem, hyperref, fancyhdr, babel, geometry

Only requirement: Docker installed.
